About the Project

How to setup

    Requirements

  • • A Computer
  • • A modded Wii

  • • 1. Download the WAD PAL WAD USA WAD
  • • 2. Install the WAD with a Wad Manager
  • Videos may take a long time to load. Try using shorter videos. If the channel randomly skips to a new video, try going back to the previous one — it should work then.
